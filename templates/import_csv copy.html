Instructions

How to Import Products:

  1. Download Template: Click the "Download Template" button to get a sample CSV file
  2. Fill Data: Open the template in Excel or any spreadsheet software and add your products
  3. Save as CSV: Save the file in CSV format (Comma Separated Values)
  4. Upload File: Click "Choose File" and select your CSV file
  5. Import: Click "Import Products" to add them to your inventory

CSV Format Requirements:

Column Name Required Description
name Yes Medicine name
manufacturer No Manufacturer name
price Yes Product price (decimal)
stock_quantity No Initial stock (default: 0)
shelf_location No Shelf location (e.g., A1, B2)
category No Product category
usage_type No Usage description
min_stock_level No Min stock threshold (default: 15)
