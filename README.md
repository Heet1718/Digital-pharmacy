## 🚀 About The Project

💊 **Digital Pharmacy Management System** is a modern pharmacy automation platform developed to simplify daily medical store operations.

It enables pharmacy owners to manage:

* 🛒 Billing & Invoicing
* 📦 Inventory Tracking
* ⏳ Expiry Monitoring
* 🚚 Supplier Management
* 👥 Customer Records
* 📊 Sales Analytics
* 📄 PDF Reports
* 💳 UPI Payments

## 🚀 Quick Start

### Clone Repository

```bash
git clone https://github.com/Heet1718/Digital-pharmacy.git
cd Digital-pharmacy
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
