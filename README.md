**MediStore Pro** is an enterprise-grade Billing, Inventory, and Business Intelligence web application engineered specifically for retail pharmacies and medical stores. Built using a robust Flask backend and an optimized MySQL relational database, the system addresses core pharmacy challenges such as precise inventory valuation, compliance-ready financial reporting, automated low-stock alerting, and real-time FIFO-based batch tracking.
